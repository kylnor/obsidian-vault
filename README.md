# Obsidian Vault

This is Kyle's personal Obsidian vault, synced via GitHub for backup and version control.

## Structure

- **youtube-notes/** - Notes from YouTube videos and tutorials
- **articles/** - Notes and summaries from articles and blog posts
- **concepts/** - Conceptual notes and idea development
- **daily-notes/** - Daily journal entries and quick notes

## About Obsidian

This vault uses [Obsidian](https://obsidian.md/) for knowledge management and note-taking.

## Sync

This repository is synced with GitHub to ensure notes are backed up and accessible across devices.
